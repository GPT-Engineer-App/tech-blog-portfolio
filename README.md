# tech-blog-portfolio

A personal website for a microcomputer technician using HTML, CSS, Svelte, SvelteKit, and Prisma. Styling frameworks include Tailwind CSS and Daisy UI. The site will feature separate sections for photos and notes, along with a home page that combines these into a blog thread.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/tech-blog-portfolio.git
cd tech-blog-portfolio
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
